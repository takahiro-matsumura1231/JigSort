# JigSort
JigSawをスワイプの入れ替え動作だけで操作可能にする

より手軽に遊ぶことができる。
ノンストレス。

強豪との差別化、女の子の画像

#工夫
JigSawのアセットをそのまま流用
操作感向上のためにインタラクション関係はイベント駆動で０から作りなおし

クリア条件を緩めると処理を簡略化できてよい

Assetの関係上、対称関係を作ったピースの生成は負荷
つまり、ソート操作に利用できない。
現在、正方形のみの対応となっている。
根本メカニズムを改善するしかない、
